ReportAnalyzer_primerCheck.py用于检测给定的一组引物在所有comibnation中出现的次数。在实际设计引物时，通常需要对每个ref seq选定2~3组候选引物。这时需要检测所有这些候选引物是否出现在一个combination中，以及在combination中出现的次数。
ReportAnalyzer_primerTag.py用于检测引物组合中具有某些标记的引物的出现次数。通常在引物末端加标记来表示某些引物特征。例如，引物名称末尾加o表示已订购的引物，加n表示新设计的引物。使用这个脚本可以方便地找出某些类别引物出现的次数。例如，找出所有combination中已订购引物出现次数最多的。
